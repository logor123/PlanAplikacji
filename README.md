Moja aplikacja bedzie dotyczyć sklepu z ciuchami będzie tam dużo różnych nieznanych marek i szeroki wybór o ich rodzaje. Plan aplikacje jest prosty:
- Prosty zachęcający interfejs by użytkownik kupywał produkty
- Sortujące filtry by było szybciej znaleźć oczekiwany produkt
- Proste opisy produktów np. z czego są stworzone lub wielkosci
- Przewodnik po rozmiarach by nikt nie miał problemu z rozmiarami ubrań
- Dodawanie produktów do koszyka i zakup w prosty szybki sposób online lub osobisty odbiór
